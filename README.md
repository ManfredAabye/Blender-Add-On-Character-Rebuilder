# CharacterRebuilder Addon für Blender 5
Dieses Plugin ist noch nicht ausgereift.

**Autor:** Manfred Aabye  
**Version:** 1.0.0  
**Kompatibel mit:** Blender 5.x  

## Zweck

Konvertiert verschiedene Rig-Typen (Rigify, Avastar, Bento Buddy, Mixamo, DAZ) in ein Second Life/OpenSim-kompatibles Bento-Rig. Unterstützt vollständiges Bone-Mapping, Preset-Management und einfache Bedienung.

## Features

- Umbenennen von Knochen nach SL/OpenSim-Bento-Standard
- Presets für Rigify, Avastar, Bento Buddy, Mixamo, DAZ
- Eigene Presets speichern/laden (.crc)
- Anpassung der Vertex Groups (Gewichte)
- Übersichtliches UI im Properties-Fenster
- Addon-Preferences für Standardpfad

## Installation

1. Blender öffnen → Edit > Preferences > Add-ons
2. "Install..." wählen und ZIP-Datei auswählen
3. Addon "CharacterRebuilder" aktivieren

## Nutzung

1. Armature auswählen
2. Im Properties-Fenster das Panel "Character Rebuilder" öffnen
3. Preset wählen (z.B. Rigify Default, Mixamo Default, DAZ Default)
4. Knochen umbenennen und gewichten
5. Optional: Preset oder Einstellungen speichern/laden

## Hinweise

- Vor Änderungen Backup anlegen
- Nicht gemappte Bones werden im UI angezeigt
- Presets können erweitert werden (Howto.md beachten)
